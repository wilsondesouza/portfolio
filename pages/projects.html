Código escrito em Python utilizando a biblioteca Custom Tkinter para a criação da interface gráfica. Sendo assim, foi necessária a implemetanção da POO. Foram utilizados dicionários para buscar e retornar valores e listas para a geração dos botões.

React

Site construído para completar Desafio do curso Descomplica + Instituto Nu. Utilizado biblioteca React para deixar o site reativo e interativo.
JavaScript também foi utilizado para aplicar algumas lógicas no Carrinho de Compras e na alteração de Temas.

Flutter

Aplicativo feito em Flutter para Android. Construído inicialmente para sanar necessidade de um comerciante local, e posteriormente adaptado para uso pessoal.
Utilizadas dependências como Provider e Shared Preferences para aproveitar informações em outras páginas e salvá-las mesmo com o reinício do aplicativo.
Outras dependências como Local Auth e Package Info foram utilizadas para implementar Biometria no sistema de log-in e para expor a versão atual do aplicativo
